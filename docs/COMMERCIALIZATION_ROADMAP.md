# 智能客服商业化落地路线

## 产品定位

本项目定位为“礼盒定制行业私域智能客服与线索转化系统”，而不是通用微信机器人。

核心商业价值：

- 秒回价格、起订量、交期、定制流程等高频问题。
- 自动识别高意向客户，沉淀到线索 CRM。
- 投诉、售后、发票、付款异常等风险场景自动转人工。
- 待人工和人工锁定会话进入接管队列，按等待时长、原因和意向分排序。
- SLA 监控首次响应、客户待回复、待回复超时和转人工超时。
- 回复安全护栏拦截绝对承诺、未核价报价、未确认排期交期等风险话术。
- 非工作时间自动发送兜底留言话术，并把客户纳入人工待处理。
- 优化待办把知识缺口、AI 兜底、人工回复和转人工复盘沉淀成可执行任务。
- 报价准备清单判断线索是否可核价，并输出缺失字段追问。
- 订单交付清单跟踪合同、付款、开票、收货地址、生产和发货状态。
- 老板和客服能看到今日咨询、待跟进线索、自动回复效果和异常状态。
- 每个客户可维护自己的产品知识、报价口径、案例和话术。

## 第一阶段：可交付 MVP

目标：能稳定部署到 1-3 个真实商家电脑，连续运行 7 天。

必须完成：

- 微信连接状态检测、断线重连和发送失败提示。
- 防重复回复、单客户限频、每日回复上限、黑白名单。
- 人工接管锁定：接管后指定时间内不自动回复，并进入接管队列和导出报表。
- 行业知识库和技能配置保持 UTF-8，可被后台或配置文件维护。
- 线索字段提取：公司、联系人、电话、微信号、数量、预算、日期、城市、节日。
- 高风险场景转人工：投诉、退款、售后、发票、付款异常、合同、法律。
- 本地日志、异常日志、最近 30 天日志清理。
- SLA 监控报告：响应达标率、超时会话、转人工超时、风险建议。
- 回复安全审计：禁用承诺、样本拦截、风险话术改写。
- 非工作时间审计：工作时间解析、当前值班状态、兜底话术。
- 优化待办：按 P0/P1/P2 输出知识库、Skills、人工话术沉淀任务。
- 报价准备：可报价率、缺失字段、建议追问和人工核价清单。
- 订单交付：合同、付款、开票、地址、生产、发货状态交接。
- 打包部署说明：依赖、启动、微信版本、异常排查。

验收标准：

- 常见问题命中率达到 80% 以上。
- 自动回复无明显重复刷屏。
- 高风险问题 100% 不直接给承诺性答案。
- 每天至少能产出可跟进线索列表。
- 客户无需改代码即可调整核心话术。

## 第二阶段：运营控制台

目标：让非技术客户能使用和维护系统。

页面模块：

- 总览工作台：今日咨询、高意向线索、待人工、自动回复率、平均响应时间。
- 会话接管：客户队列、聊天记录、AI 决策、人工锁定、等待时长、接管原因、下一步建议。
- 风控质检：SLA 达标率、待回复超时、人工接管超时、回复安全护栏、非工作时间兜底、优化待办、知识缺口、操作审计。
- 线索 CRM：报价准备率、缺失字段、建议追问、人工核价清单。
- 订单交付：订单完整率、待补充字段、生产和发货状态。
- 线索 CRM：阶段、负责人、意向分、预算、数量、日期、城市、下一步动作。
- 知识库：产品、价格、起订量、交期、物流、售后、案例。
- Skills：报价、交期、案例、线索收集、风控转人工。
- 系统设置：微信状态、AI 接口、模型优先级、黑白名单、回复阈值。

验收标准：

- 客户能新增和修改知识条目。
- 客户能查看每条自动回复的命中依据。
- 客服能一键接管和释放会话。
- 老板能导出待跟进线索。

## 第三阶段：商业闭环

目标：证明系统带来订单转化，而不只是减少人工。

关键能力：

- 线索阶段流转：新咨询、已留资、已报价、方案确认、打样、生产、成交、流失。
- 自动跟进提醒：超过指定时间未处理的高意向客户进入待办。
- 质检闭环：未命中问题、人工改写话术、错误回复标记。
- 知识库缺口发现：按高频未命中问题生成补充建议。
- 成本统计：AI 调用次数、失败率、延迟、备用模型切换次数。
- 周报：咨询量、线索数、成交金额、流失原因、人工接管原因。

验收标准：

- 能按客户/日期/阶段追踪线索。
- 能看到自动回复带来的留资数量。
- 能复盘流失原因并反哺话术。
- 能证明节省客服时间或提升线索响应率。

## 第四阶段：多渠道与 SaaS 化

目标：从单机交付升级为可复制商业产品。

优先级：

1. 微信 PC 自动化继续作为私域 MVP 入口。
2. 增加微信客服 API 或企业微信客服作为合规入口。
3. 增加公众号、小程序、网页客服入口。
4. 服务端统一会话、线索、知识库和报表。
5. 多租户、权限、计费、审计、备份恢复。

验收标准：

- 每个客户的数据、配置、知识库隔离。
- 支持按客户开通/停用模型和渠道。
- 支持按月续费和交付版本升级。

## 推荐收费模型

- 试点版：199-399 元/月，单微信、基础知识库、基础线索表。
- 标准版：699-1299 元/月，控制台、CRM、人工接管、周报。
- 定制版：3000-10000 元部署费 + 月维护费，含行业话术和知识库定制。
- 代运营版：系统 + 话术优化 + 数据复盘，按月 2000 元以上。
