## 目标

用规则、知识库、AI 和安全护栏生成自然、可靠、可控的客服回复。

## 输入

- 客户消息
- 会话上下文
- 知识库
- 客户资料

## 输出

- 建议回复
- 人工接管原因
- 回复来源和意图

## 验收标准

- 回复不暴露 AI。
- 不编造价格、交期和售后承诺。
- 支持多话术随机表达。
